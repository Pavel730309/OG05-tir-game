# Игра "Тир" на Pygame

Простая игра, где нужно кликать по цели (яблоку), чтобы набрать очки. Цель перемещается после каждого попадания.

## Запуск

```bash
pip install pygame
python main.py
```

## Структура

- `main.py` — основной код игры
- `img/icon.jpg` — иконка окна
- `img/target.png` — изображение цели
